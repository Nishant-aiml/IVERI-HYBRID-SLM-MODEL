# Stage 3B — Proprietary Indian CS & Engineering Dataset Scaffold

This directory stores proprietary university question papers, GATE questions, syllabus concept QA, and computer science placement interview Q&A.

## Subdirectories

- `university_papers/`: Semantic question paper exports from Anna University, VTU, AKTU, and other major tech institutions (years 2015-2025).
- `gate_questions/`: GATE Computer Science exam questions (years 1991-2025) with detailed step-by-step mathematical answers.
- `placement_qa/`: CS placement interview Q&A (data structures, algorithms, system design, DBMS, OS, networking).
- `subject_explanations/`: Syllabusconcept explanations matching standard semester curriculums.

See **`FORMAT.md`** for the JSON schema and **`python scripts/ingest_stage3b.py`** for ingestion.
