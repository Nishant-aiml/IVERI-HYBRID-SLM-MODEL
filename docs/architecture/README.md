# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the IVERI project.

## Format

Each ADR follows this structure:

```
## ADR-NNN: Title

**Date:** YYYY-MM-DD
**Status:** Proposed | Accepted | Deprecated | Superseded
**Phase:** Phase X

### Context
Why this decision is needed.

### Decision
What was decided.

### Consequences
What changes as a result.
```
